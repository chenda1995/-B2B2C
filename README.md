1.配置好主机，将文件放到配置好主机的根目录下。  
  
  
2.修改根目录下 config.inc.php 文件  
	|--$CFG['db_host'] = '';		数据库地址  
	|--$CFG['db_name'] = '';       		数据库名   
	|--$CFG['db_user'] = '';       		数据库帐号   
	|--$CFG['db_pass'] = '';       		数据库密码  
   
3.请在 https://99ruyuan.top/public/Data.sql 本链接下载Data.sql文件，数据库文件导入至项目数据库中 
  
4.做好以上配置打开 http：//你的网址/admin.php 登陆后台：  
	|--user：admin  
	|--password：123456
